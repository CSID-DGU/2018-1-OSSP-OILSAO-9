## 실행 방법

1. 리눅스 환경에서 dodge 폴더를 다운로드
2. 리눅스 터미널에서 닷지를 다운 받은 directory로 이동
3. 터미널에서 dodge 폴더 경로에서 MakeFile에 있는 명령어를  입력 및 실행
4. 닷지 폴더 내에 생성된 main파일 실행
5. 게임 플레이!



## 실행 시 유의사항


- 실행 시, mysql 설치 되어 있어야함
- 게임 플레이시, 마우스를 움직이면 오류가 날 수 있으니 마우스를 움직이지 않는 것을 권장
- 멀티 모드 관련 유의사항은 1차 수정 소스 깃헙 참고
